# Contributing to MasterHttpRelayVPN GUI

Thank you for wanting to help! Here is how you can contribute:

## Reporting Bugs
Open an issue on the [Issues tab](../../issues) with:
- Your operating system (Windows 10/11, Ubuntu, etc.)
- The steps to reproduce the bug
- What you expected to happen vs what happened
- Any error messages or log output

## Suggesting Features
Open an issue and describe the feature you'd like to see.

## Pull Requests
1. Fork the repository (click Fork button at the top of the page)
2. Make your changes in your fork
3. Open a Pull Request explaining what you changed and why

## Translations
Translations of the README or the built-in guide are very welcome!
Especially Persian (Farsi), Russian, Chinese, and Arabic.
